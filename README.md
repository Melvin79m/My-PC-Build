# Professional PC Build Project — From Research to Assembly

## Overview
This project documents the complete end-to-end process of building a custom desktop PC designed for IT support, virtualization, system administration practice, and networking fundamentals. The goal of this build is to demonstrate practical hardware knowledge, component selection reasoning, troubleshooting workflow, and hands-on experience assembling a reliable workstation for IT career development.

## Build Objectives
- Create a stable and efficient workstation for labs, ticket simulations, and certification study.
- Learn and document each hardware component, why it was selected, and how it contributes to system performance.
- Demonstrate real-world troubleshooting, BIOS configuration, OS installation, and system optimization.
- Showcase hardware competency for IT support and helpdesk roles.

# Why I Selected Each Component

## CPU: AMD Ryzen 7 9700X
Chosen for strong multi-core performance and energy efficiency, ideal for running VMs, lab environments, and multitasking during IT troubleshooting practice.

## AIO Cooler: Thermalright Grand Vision 360 with LCD
Provides excellent thermal control with low noise. The LCD panel gives real-time CPU data, reinforcing monitoring and maintenance skills.

## Motherboard: Gigabyte B850 Gaming X WiFi 6E
Selected for AM5 future-proofing, built-in Wi-Fi 6E, and plenty of connectivity for IT tools, flash drives, external drives, and network adapters.

## GPU: Gigabyte Radeon RX 9060 XT
Ensures smooth virtualization, multi-monitor setups, and hardware acceleration during workloads.

## RAM: 32GB DDR5 6000MHz (EXPO)
Allows running multiple virtual machines for practicing Active Directory, ticket simulations, networking labs, and both Linux and Windows environments.

## Dual Samsung 990 NVMe SSDs
Each drive dedicated to its own OS:
- Windows 11 for IT support workflows and Microsoft ecosystem practice
- Ubuntu/Kali for Linux system administration and networking labs

## PSU: MSI MAG A750GL (80+ Gold)
Reliable and stable power delivery with PCIe 5.0 support to cover future GPU upgrades.

## Case: Antec C5 ARGB
Airflow-optimized, easy to cable-manage, and ideal for clean layout and troubleshooting.

## GPU Support Bracket
Prevents GPU sag and includes a digital temp display for monitoring and diagnostics.

# Project Breakdown

## 1. Hardware Research & Selection
Researched compatibility, VRM quality, airflow requirements, upgrade paths, Linux/Windows support, and overall reliability. The goal was to understand how each part works together from an IT support perspective.

## 2. Full PC Assembly Process
Documented steps include:
- Installing CPU, RAM, and AIO
- Mounting and routing AIO tubes
- Applying thermal paste
- Installing PSU and cable management
- Mounting NVMe SSDs
- Installing motherboard into the case
- Connecting I/O headers, fans, and RGB
- Installing GPU and configuring support bracket
- First boot, BIOS check, and troubleshooting

## 3. BIOS Configuration
Applied:
- EXPO profile for RAM
- Virtualization (SVM)
- Boot priority for OS installs
- Fan curve adjustments
- Temperature and stability verification

## 4. Dual-Boot OS Setup

### Drive 1 — Windows 11
Used for:
- IT support tools
- PowerShell practice
- Ticket simulations
- Active Directory labs
- Helpdesk workflows

### Drive 2 — Ubuntu / Kali Linux
Used for:
- Linux system administration
- Networking tools
- Bash scripting
- Package management practice

Configured BIOS boot selection and verified both drives operate independently.

## 5. Post-Build Testing & Troubleshooting
Performed:
- MemTest
- CPU/GPU stress testing
- Fan tuning
- Driver updates
- OS hardening steps
- Monitoring with AIO LCD and GPU bracket display
